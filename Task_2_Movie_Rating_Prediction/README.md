# Movie Rating Prediction

## Overview
This project is part of the CodSoft Data Science Internship.

The objective of this project is to build a machine learning model that predicts movie ratings based on genre, director, and actors.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Model
- Random Forest Regressor

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Label Encoding
4. Model Training
5. Prediction
6. Model Evaluation
7. Data Visualization

## Result
The model successfully predicts movie ratings using machine learning techniques.
